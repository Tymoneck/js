1) Kiedy powstał JavaScript (podaj datę).

JavaScript został stworzony w 1995 roku.

2) Dlaczego jest to język skryptowy

Nie musi zostać skompilowany do kodu maszynowego, aby można było zobaczyć efekty jego działania.

3) Gdzie jest wykonywany JS (klient lub serwer)

JS jest wykonywany na kliencie

4) Dlaczego jest JS to bezpieczne narzędzie

JavaScript to bezpieczne narzędzie, ponieważ ma znacznie ograniczone uprawnienia dostępu do zasobów komputera.

5) Czy jest to język obiektowy , jeśli tak to dlaczego.

Javascript jest językiem programowania obiektowego ponieważ wykorzystuje definicje obiektów, metod, zdarzeń, które mogą być zastosowane do opisu właściwości obiektów.

6) Wymień zalety JS:

Zaleta1: Prostota JavaScript. Nic nie stoi na przeszkodzie, aby zacząć używać go już po kilku minutach od rozpoczęcia nauki.
Zaleta2: Z pomocą frameworków (rodzaj programu do tworzenia innych programów w JS) zostały stworzone znane na całym świecie aplikacje, np. Google używa Angular, a Facebook — React.
Zaleta3: Pozwala on na odciążenie serwerów i ograniczenie ilości zbędnych danych przesyłanych pomiędzy nimi a przeglądarką klienta.

7) Framework to:

rodzaj programu do tworzenia innych programów w JS.

8) Do aplikacji mobilnych można użyć Framework

React Native.

9) Aplikacje responsywne to:

Responsywność JavaScript udostępnia metody, których można użyć w aplikacji, aby uzyskać bieżący rozmiar ekranu i wykorzystać wynik do wykonania działań w oparciu o ten rozmiar. Używając CSS i HTML można zmienić rozmiar, ukrywać, zmniejszać, powiększać lub przenosić treści, aby wyglądały dobrze na dowolnym ekranie.

10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
Powyższy fragment kodu tworzy przycisk, który przywołuje funkcję WinOpen().

11) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Fragment kodu otwiera nowe okno, w którym wyświetla się plik obraz.html

toolbar=no -> ukrywa przyciski katalogów
directories=no, -> wytłumaczenie
menubar=no, -> wytłumaczenie
height=280, -> ustawia wysokość okna
width=160, -> ustawia szerokość okna
top=200, -> ustawia odległosć okna od góry ekranu
left=200 -> ustawia odległosć okna od lewej strony ekranu

12) Opisz następującą instrukcję window.close()

Instrukcja windows.close() zamyka okno przeglądarki
