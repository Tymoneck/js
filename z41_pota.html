Dział 6 Pętle - Rodzaje pętli

1) Pętla for

Konstrukcja: 
for (zainicjowanie_zmiennej; warunek_kończący_wykonywanie_pętli; zmiana_zmiennej) {
    kod który zostanie wykonany pewną ilość razy
}
Opis:

Pętla for wykonuje blok kodu określoną liczbę razy. Zawiera inicjalizację zmiennej, warunek zakończenia oraz zmianę zmiennej w każdej iteracji.

2) Pętla while (z kontrolowanym wejściem)

Konstrukcja: 
while (wyrażenie_sprawdzające_zakończenie_pętli) {
    ...fragment kodu który będzie powtarzany...
}
Opis:

Pętla while najpierw sprawdza warunek, a następnie wykonuje blok kodu. Pętla może się nie wykonać, jeśli warunek jest fałszywy od początku.

3) Pętla do while (z kontrolowanym wyjściem)

Konstrukcja: 
do {
    ...fragment kodu który będzie powtarzany...
} while (warunek);
Opis:

Pętla do...while zawsze wykona przynajmniej jedną iterację, zanim sprawdzi warunek zakończenia.
